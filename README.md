# 🎮 OnlyGoes - CS2 Autoconfig Tool

Ferramenta automatizada em PowerShell para provisionamento rápido e gerenciamento de configurações do Counter-Strike 2. Ideal para padronização de ambientes, LAN Houses e formatações limpas.

## 🚀 Como Usar (Execução Fileless)

Não é necessário baixar nenhum arquivo manualmente. Abra o **PowerShell** e execute o comando abaixo:

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; irm [https://cs2.onlygoes.com.br](https://cs2.onlygoes.com.br) | iex
```

## ⚙️ Funcionalidades

- **Instalação Silenciosa da Steam**: Instala o cliente Steam sem a necessidade de interação do usuário.
- **Restauração de Configurações**: Restaura automaticamente o arquivo `autoexec.cfg` e outras configurações do CS2.
- **Backup Inteligente**: Faz backup das configurações existentes antes de restaurar as novas.
- **Interface Amigável**: Interface simples e intuitiva para facilitar o uso.

## 🔧 Requisitos

- **Windows 10** ou superior.
- **PowerShell** versão 5.0 ou superior.
- **Conexão com a Internet** para baixar os arquivos necessários.
